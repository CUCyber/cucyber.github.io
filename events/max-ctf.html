Max's Capture The Flag
East Coast Hospital Security Assessment

Jul 7, 2017 - Jul 15, 2017 Max Harley Events

Greetings member of CU Cyber,

We received an email from the IT department at East Coast Hospitals that we would like you to check out:

Hello! My name is Dan and I'm so glad we hired you to do our internal security audit. My job is to do IT around the hospital. I had to set up every server in this place as well as do most of the programming. I'm not a programmer, so you may find a few security issues there, but it can't be too bad, can it? I don't have a network diagram to give you, but I'll let you figure it out.

They would like us to do some penetration testing on their environment, so we need your help to find the vulnerabilities. The goal of the competition is to exploit vulnerabilities in their applications to find "flags" that you can submit to a scoreboard to gain points. To participate, you'll need to get OpenVPN credentials to the hospital network. You'll probably also want to install the recommended tools below. Support for getting connected or inquiries is available at our email, cyber@clemson.edu, and via our Discord server, https://discord.gg/f7ZcC7z, channel #ctf.

Rules

  • Scoreboard is available at scoreboard.cyber (10.13.37.2)
  • The scoreboard is out-of-bounds
  • The firewall (10.13.37.1) is out-of-bounds
  • Port 2222 on any of the machines is out-of-bounds
  • Disabling running services is out-of-bounds
  • More rules may be added later - These will be announced

Getting Credentials

Email memes and the rules to cyber@clemson.edu from your Clemson email and we will email you back a .ovpn file to connect to the competition network. Put "Max's CTF" in the subject. If you aren't comfortable sending memes, you're allowed to only send the rules.

Connecting to the Competition Environment

Linux

NetworkManager (use this if you are unsure):
  1. Install NetworkManager-openvpn-gnome (Fedora) or network-manager-openvpn-gnome (Ubuntu)
  2. Pull up your edit connections dialog
  3. Add connection of type VPN
  4. Select import from file and give it the .ovpn file emailed to you.
  5. Connect to the VPN from the applet
CLI:
  1. Install openvpn
  2. Run sudo openvpn --config <emailed.ovpn>

macOS

  1. Install Tunnelblick from https://tunnelblick.net/
  2. Select the .ovpn file emailed to you
  3. Connect to the VPN from the applet

Windows

  1. Install OpenVPN from http://openvpn.net/index.php/open-source/downloads.html
  2. Select the .ovpn file emailed to you
  3. Connect to the VPN from the applet

Tools You Will Want

Linux

  • Burp Suite (https://portswigger.net/burp/), Tamper Chrome (https://github.com/google/tamperchrome), mitmproxy, or any such tool you are comfortable with
  • ncat - it's like netcat, but way better (e.g. supports TLS, shells, telnet control characters, converting LF to CRLF)

macOS

  • Homebrew (https://brew.sh/)
  • Burp Suite (https://portswigger.net/burp/), Tamper Chrome (https://github.com/google/tamperchrome), mitmproxy, or any such tool you are comfortable with
  • ncat - it's like netcat, but way better (e.g. supports TLS, shells, telnet control characters, converting LF to CRLF)

Windows

  • KiTTY (http://www.9bis.net/kitty/?page=Download)
  • VirtualBox (https://www.virtualbox.org/wiki/Downloads)
  • Kali Linux VM (https://www.kali.org/ - Install in VirtualBox)
